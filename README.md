My Assignment Project - User Management System Assignment-Project This is my Assignment project - User Management System A modern React.js project for managing users. Fetches initial users from an API and allows adding, editing, viewing, and deleting users using localStorage. Fully responsive with smooth hover and 3D effects.

Features

✅ Fetch users from API (https://jsonplaceholder.typicode.com/users)

✅ Add new users and save them in localStorage

✅ Edit existing users

✅ View user details

✅ Delete users with confirmation

✅ Responsive design for desktop and mobile

✅ Modern UI with 3D hover effects and smooth transitions

✅ Notifications using React Toastify

Pages

Home – List all users with Edit, Delete, and View actions.

Create User – Add a new user with name, email, and phone.

Edit User – Edit existing user details.

User Details – View full details of a selected user.

Technologies Used

React.js (Functional Components & Hooks)

React Router DOM

Tailwind CSS for styling

React Toastify for notifications

LocalStorage for data persistence
